# Kambio Roadmap

## Phase 1: Project Setup

- Scaffold React + Vite app.
- Add styling system.
- Set up folder structure.
- Add mock data files.

## Phase 2: Mobile App Shell

- Build responsive mobile layout.
- Add top header with credit balance.
- Add bottom navigation.
- Define app routes or tab-based sections.

## Phase 3: Core Screens

- Dashboard overview.
- Skill marketplace.
- Skill detail/request flow.
- Offer skill form.
- Earn credits/tasks screen.
- Wallet and transaction history.

## Phase 4: Prototype Interactions

- Add temporary skill listing.
- Request a skill session.
- Deduct credits when spending.
- Complete tasks to earn credits.
- Add transaction history records.

## Phase 5: Polish

- Improve spacing, typography, and visual hierarchy.
- Test mobile viewport layouts.
- Add empty/success states.
- Prepare presentation walkthrough.

## Phase 6: Optional Enhancements

- Add localStorage persistence.
- Add category filters.
- Add profile screen.
- Add onboarding slides.
- Add simple animations.
