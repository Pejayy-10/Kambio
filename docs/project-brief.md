# Kambio Project Brief

Kambio is a high-fidelity frontend prototype for an asynchronous skill exchange platform.

The platform solves a classic bartering limitation: two people do not need to directly want each other's skills at the same time. Instead, users earn skill credits by teaching or completing micro-tasks, then spend those credits to learn from others.

## Prototype Goal

Create a polished mobile-first UI that demonstrates:

- A skill marketplace
- A credit-based exchange system
- A user dashboard
- Temporary add/request/complete interactions
- Frontend-only state simulation
- Delivery models for async help, video calls, face-to-face tutorials, and mini courses
- Progress tracking for requested and active exchanges
- Teacher rewards beyond credits, including reputation, badges, and portfolio proof

## Non-Goals

- No backend
- No real authentication
- No database
- No payment system
- No production deployment requirement yet

## Target Audience

This prototype is built for a thesis/class presentation and should clearly communicate the product concept, user flows, and interface quality.
